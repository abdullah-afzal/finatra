Scrooge and Finagle¶

Finatra’s thrift support is meant to be used in conjunction with Finagle clients and is a Finagle server.

Thus it is expected that you will be using Scrooge to generate Finagle thrift bindings.

Finatra is a Scala services framework built on top of TwitterServer and Finagle.
